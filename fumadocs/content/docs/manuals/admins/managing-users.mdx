As an admin, managing users on SEMS involves overseeing user accounts, roles, and permissions across the platform.

<Callout title="Prerequisites">
    You must be a SEMS super administrator to access user management features on SEMS.  
    Ensure you have the necessary permissions before proceeding.
</Callout>

### Accessing User Management

Go to the SEMS User Management Page under the Admins tab in the navigation menu.

![SEMS User Management](/images/SEMS_Admin_Managing_Users_1.png)

### What You Can Do

Admins can manage user accounts, roles, and permissions across the platform.
